Tes Karakteristik Pribadi (TKP)

1. Menurut orang lain kedisiplinan dan rasa tanggung jawab saya...

A. Sangat baik
B. Baik
C. Cukup
D Kurang
E. Rendah

Skor: A=5, B=4, C=3, D=2, E=1

2. Rekan-rekan kerja saya sering datang terlambat 20 menit setelah aturan jam kerja.

A. Saya mengikuti teman-teman saya.
B Saya mengetahui aturan jam kerja kantor.
C. Saya datang sesuai jam kerja kantor karena saya takut dimarahi pimpinan.
D. Saya datang bekerja sesuai dengan jam
E. Saya ingin saya mendapat-kan predikat karyawan yang rajin bekerja jika saya berangkat sesuai jam kantor.

Skor: A=1, B=4, C=3, D=1, E=2

3. Jika ada gosip atau informasi yang belum ada kejelasannya, saya

A. Saya mengkonfirmasi informasi tersebut berdasarkan fakta.
B. Saya tetap mempercayainya.
C. Saya tidak peduli.
D. Saya ragu-ragu untuk mempercayainya.
E. Saya percaya jika banyak orang yang mempercayai informasi tersebut

Skor: A=5, B=2, C=1, D=3, E=2

4. Jika saya mendapatkan hadiah maka ....

A. Saya akan merasa senang.
B. Saya akan membagi kebahagiaan saya dengan bersedekah.
C. Saya akan menolaknya karena saya berkecukupan.
D. Saya menerima dan saya gunakan sendiri.
E. Saya tidak peduli dengan orang lain.

Skor: A=4, B=5, C=3, D=3, E=1

5. Dalam melaksanakan pekerjaan sehari-hari saya adalah tipe pekerja yang

A. Senang menyelesaikan semua tanggung jawab sebelum waktunya.
B. Menyelesaikan pekerjaan mendekati masa akhir deadline.
C. Bekerja dengan hati-hati dan tepat waktu.
D. Melakukan pekerjaan sedikit demi sedikit sampai tuntas.
E. Menyelesaikan pekerjaan dengan sangat cepat.

Skor: A=4, B=1, C=5, D=3, E=2
